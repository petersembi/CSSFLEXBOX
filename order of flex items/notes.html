- according to flex, the first item is in order 0. - we can change the order of arrangement of items by using order property. - we can grow the size of flex-items by using flex-grow. - to grow the size of a specific flex item, use flex-grow property. it will be x times as large as the other items. - flex basis: when using the flex-basis property, we should use percentage value or pixel value flex-basis is the same as the width property. - - we use the flex shrink property to determine whether the flex-item will shrink or not even after specifying pixel size. flex-wrap: by defualt its value is nowrap. changing it to wrap, will ensure that it takes anything overflowing to the next line.
